DocBooker Frontend:-

Patient-side application for the DocBooker Hospital Management System.

Features:

- Patient Registration & Login
- Book Appointment
- Cancel Appointment
- Reschedule Appointment
- Appointment History
- Responsive UI
- JWT Authentication

Tech Stack:

- React.js
- Axios
- React Router DOM
- React Toastify
- CSS

Live Demo:

Frontend: 
https://doc-booker-frontend.vercel.app

Backend API:
https://docbooker-backend-2hzo.onrender.com

Installation:

npm run dev
